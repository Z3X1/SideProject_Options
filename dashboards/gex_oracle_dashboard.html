市場指標
DVOL（Deribit Volatility Index）
Deribit 計算的 BTC 30天隱含波動率指數，反映市場對未來波動的預期定價。DVOL 越高代表市場越不確定。
FR（Funding Rate）資金費率
永續合約多空雙方的定期支付費率。FR > 0% 多方付空方（多頭過熱），FR < 0% 空方付多方（空頭主導）。8h 週期。
L/S（Long/Short Ratio）多空比
持有多單的帳戶數量 / 持有空單的帳戶數量。L/S > 2 表示空頭超配，為逆向指標。
OI（Open Interest）未平倉量
目前市場上尚未結算的合約總量。OI 持續上升 + FR 轉向 = 強方向性信號。
期權術語
GEX（Gamma Exposure）Gamma 暴露
造市商整體 Gamma 頭寸的加總。正 GEX = 造市商是穩定器（反向對沖）；負 GEX = 造市商是放大器（同向對沖）。
Gamma Flip
GEX 從正轉負的臨界價格。Spot 在上方（POS Regime）時市場相對穩定；跌破後（NEG Regime）波動加劇。
PCR（Put/Call Ratio）
Put OI / Call OI。PCR > 1 表示 Put 為主（偏防禦），PCR < 1 表示 Call 為主（偏進攻）。需結合跨期結構分析。
Call Wall / Put Wall
特定行使價上 Call/Put OI 集中的位置。Call Wall 形成上方阻力，Put Wall 提供下方支撐（ATM 時最強）。
GEX 框架術語
Gamma-MP（Gamma Magnet Point）
Gamma 集中度最高的價格，對 Spot 具有磁吸效應。造市商 Delta 對沖行為使 Spot 傾向向此靠攏，是結算中位預測的核心錨點。
POS Regime / NEG Regime
POS（正 Gamma 體制）：Spot > Gamma Flip，造市商穩定器，波動自我抑制。NEG（負 Gamma 體制）：Spot < Gamma Flip，造市商放大器，波動自我強化。
SS（Short Squeeze / Shock Scenario）
框架中指深跌情境（Shock Scenario）。充分條件：FR 深負 + 催化劑 + 突破 Gamma Flip + 足夠時間。P(SS) 用乘積法計算。
Layer1 / Layer2
Layer1：GEX 結構直接推導的阻力/支撐。Layer2：技術指標（EMA、MACD 等）的廣阻力帶。POS Regime 下兩層合併計算；NEG Regime 下嚴格分離。
技術指標
MACD（DIF / DEA / Histogram）
DIF = 短期EMA - 長期EMA，DEA = DIF 的信號線。DIF < DEA = 死叉（看跌），DIF > DEA = 金叉（看漲）。框架要求信號持續時間達到壽命標準才算有效。
RSI（Relative Strength Index）
相對強弱指標，範圍 0-100。RSI < 30 超賣，RSI > 70 超買。框架區分「機械滾動（×0.5）」和「真實超賣（×1.0）」兩種效果。
EMA（Exponential Moving Average）
指數移動平均線。框架關注 EMA5/10/30/200。EMA200(4h) 與 EMA30(2h) 之間的密集帶構成 Layer2 廣阻力帶。
